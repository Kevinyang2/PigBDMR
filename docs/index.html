Model pipeline

From query to behavior moments.

The page mirrors the released inference output: each query produces ranked temporal windows with confidence scores, while the evaluator computes mAP, mIoU, and recall metrics from the same prediction format.

1

Feature extraction

Video and text encoders produce surveillance behavior tokens and query representations.

2

Virtual alignment

Adaptive cross-attention aligns video tokens, language tokens, and learnable virtual tokens.

3

Temporal prediction

Multi-scale heads produce candidate behavior windows for single or repeated moments.

4

Verification signal

Training-time proposal verification improves boundary quality while keeping inference lightweight.
