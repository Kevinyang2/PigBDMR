PigBDMR Qualitative Demo

Representative multi-moment retrieval results for three Pig-MMR2 queries. Each case shows the video filmstrip, ground truth, and our predicted temporal moments on the same axis.

GitHub README

Ours uses matched predictions from the released PigBDMR submission format, following the repository visualization script: candidate windows are matched to ground-truth moments by temporal IoU, then displayed as P1/P2/P3 intervals.
